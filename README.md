# tannlege-irgens-hov.no

Website for dentist Marit Irgens Hov. Static HTML/CSS site hosted on GitHub Pages at [tannlege-irgens-hov.no](https://tannlege-irgens-hov.no).

## Structure

- `index.html` — single page with sections for about, services, booking, and contact
- `styles.css` — all styles
- `script.js` — scroll animations
- `marit.webp` — optimized hero photo used on the page
- `marit.jpg` — retained for metadata/social sharing compatibility

## Deploy

Push to `main`. GitHub Pages publishes automatically.
